Discrete Fourier Transform

Let us define DFT as follows:

X(k)=1N∑j=0N−1x(j)ωNjk \begin{equation} X(k) %=\textrm{DFT}_N[x](k) =\frac{1}{\sqrt{N}}\sum_{j=0}^{N-1}x(j)\omega_N^{jk} \end{equation} X(k)=N​1​j=0∑N−1​x(j)ωNjk​​​

where N=2n, ωN=ei2πN, x(j)N=2^n,~\omega_N = e^{i\frac{2\pi}{N}},~x(j)N=2n, ωN​=eiN2π​, x(j) is the nnn-th element of the input vector xxx and X(k)X(k)X(k) is the kkk-th element of the transformed vector. In DFT, it is implicitly assumed that the input vector cycles back after NNN data points. i.e.
x(j+N)=x(j)x(j+N)=x(j)x(j+N)=x(j)

The goal of QFT is to build a circuit that transforms an amplitude encoded vector. If ∣x⟩\ket{x}∣x⟩ is a nnn qubit state, then equation (1)(1)(1) can be written as follows:

⟨k∣QFT∣x⟩=1N∑j=0N−1⟨j∣x⟩ωNjk \braket{k|\textrm{QFT}|x} = \frac{1}{\sqrt{N}} \sum_{j=0}^{N-1}\braket{j|x}\omega_N^{jk} ⟨k∣QFT∣x⟩=N​1​j=0∑N−1​⟨j∣x⟩ωNjk​

where k∈ZNk\in\Z_Nk∈ZN​. Taking, x=l∈ZNx=l\in\mathbb{Z}_Nx=l∈ZN​ we can build up the QFT\textrm{QFT}QFT matrix since ⟨k∣QFT∣l⟩\braket{k|\textrm{QFT}|l}⟨k∣QFT∣l⟩ is the element on the kkk-th row and lll-th column of the QFT\textrm{QFT}QFT matrix.

⟨k∣QFT∣l⟩=1N∑j=0N−1⟨j|l⟩ωNjk=1N∑j=0N−1δjlωNjk=ωNlkN\def\bra#1{\left\langle#1\right|}\def\ket#1{\left|#1\right\rangle} \def\braket#1#2{\left\langle#1\middle|#2\right\rangle} \bra{k}\textrm{QFT}\ket{l} = \frac{1}{\sqrt{N}}\sum_{j=0}^{N-1}\braket{j}{l}\omega_N^{jk} = \frac{1}{\sqrt{N}}\sum_{j=0}^{N-1}\delta_{jl}\omega_N^{jk} = \frac{\omega_N^{lk}}{\sqrt{N}}⟨k∣QFT∣l⟩=N​1​j=0∑N−1​⟨j∣l⟩ωNjk​=N​1​j=0∑N−1​δjl​ωNjk​=N​ωNlk​​

If we write it as a matrix in the computational basis, we get,

QFT=1N(111…11ωNωN2…ωNN−11ωN2ωN4…ωN2(N−1)⋮⋮⋮⋱⋮1ωNN−1ωN2(N−1)…ωN(N−1)2)\textrm{QFT} = \frac{1}{\sqrt N}\begin{pmatrix} 1&1&1&\dots&1\\ 1&\omega_N&\omega_N^2&\dots&\omega_N^{N-1}\\ 1&\omega_N^2&\omega_N^4&\dots&\omega_N^{2(N-1)}\\ \vdots&\vdots&\vdots&\ddots&\vdots\\ 1&\omega_N^{N-1}&\omega_N^{2(N-1)}&\dots&\omega_N^{(N-1)^2} \end{pmatrix}QFT=N​1​⎝⎛​111⋮1​1ωN​ωN2​⋮ωNN−1​​1ωN2​ωN4​⋮ωN2(N−1)​​………⋱…​1ωNN−1​ωN2(N−1)​⋮ωN(N−1)2​​⎠⎞​

This is of course, same as the DFT matrix (maybe except for the normalization factor). One of the important property of this matrix is that it is an unitary matrix and thus it is possible to build a quantum circuit for it.

Fast Fourier Transform

Directly computing DFT by multiplying the input vector with the DFT matrix has a time complexity of O(N2)O(N^2)O(N2). However, there is a more efficient algorithm commonly referred to as FFT. It is a divide and conquer algorithm and has a time complexity of O(Nlog⁡N)O(N\log N)O(NlogN). Let us discuss how FFT improves over DFT before we move onto QFT.
Here we are going to explain radix-2 decimation-in-time1 FFT algorithm in details.

Let us define, xEx_ExE​ to be the vector that is composed of the even half of the input vector xxx and xOx_OxO​ to be the vector that is composed of the odd half of the input vector. i.e.
xE(j)=x(2j)∀j∈ZN2xO(j)=x(2j+1)∀j∈ZN2 \begin{align*} x_E(j) &= x(2j)& \forall j\in %\left[0,\tfrac{N}{2}\right) \Z_\frac{N}{2} \\ x_O(j) &= x(2j+1)& \forall j\in %\left[0,\tfrac{N}{2}\right) \Z_\frac{N}{2} \end{align*} xE​(j)xO​(j)​=x(2j)=x(2j+1)​∀j∈Z2N​​∀j∈Z2N​​​

Both xEx_ExE​ and xOx_OxO​ has a length of N2\frac{N}{2}2N​. And since xxx has a period of NNN, both xEx_ExE​ and xOx_OxO​ has a period of N2\frac N22N​. Let XEX_EXE​ and XOX_OXO​ to be the DFT applied output vector of xEx_ExE​ and xOx_OxO​ respectively.

XE(k)=1N2∑j=0N2−1xE(j)ωN2jkXO(k)=1N2∑j=0N2−1xO(j)ωN2jk \begin{align*} X_E(k) &= \frac{1}{\sqrt{\frac{N}{2}}}\sum_{j=0}^{\frac{N}{2}-1}x_E(j)\omega_\frac{N}{2}^{jk}\\ X_O(k) &= \frac{1}{\sqrt{\frac{N}{2}}}\sum_{j=0}^{\frac{N}{2}-1}x_O(j)\omega_\frac{N}{2}^{jk}\\ \end{align*} XE​(k)XO​(k)​=2N​​1​j=0∑2N​−1​xE​(j)ω2N​jk​=2N​​1​j=0∑2N​−1​xO​(j)ω2N​jk​​

Also notice that, ωN2=(ei2πN)2=ei2πN2=ωN2\omega_N^2 = \left(e^{i\frac{2\pi}{N}}\right)^2=e^{i\frac{2\pi}{\frac{N}{2}}} = \omega_\frac{N}{2}ωN2​=(eiN2π​)2=ei2N​2π​=ω2N​​ and ωNN2=ei2πN⋅N2=eiπ=−1\omega_N^{\frac N2} = e^{i\frac{2\pi}{N}\cdot\frac N2} = e^{i\pi}=-1ωN2N​​=eiN2π​⋅2N​=eiπ=−1.

Now, we can rewrite equation (1)(1)(1) as follows:
X(k)=1N∑j=0N−1x(j)ωNjk=1N∑j∈evenx(j)ωNjk+1N∑j∈oddx(j)ωNjk=1N∑j=0N2−1x(2j)ωN2jk+1N∑j=0N2−1x(2j+1)ωN(2j+1)k=1N∑j=0N2−1xE(j)ωN2jk+1N∑j=0N2−1xO(j)ωN2jkωNk[∵xE,xO]=1N∑j=0N2−1xE(j)ωN2jk+ωNkN∑j=0N2−1xO(j)ωN2jk[∵ωN2=ωN2]=12⋅N2∑j=0N2−1xE(j)ωN2jk+ωNk2⋅N2∑j=0N2−1xO(j)ωN2jk=121N2∑j=0N2−1xE(j)ωN2jk+ωNk21N2∑j=0N2−1xO(j)ωN2jk=12XE(k)+ωNk2XO(k) \begin{align*} X(k) &= \frac{1}{\sqrt{N}}\sum_{j=0}^{N-1}x(j)\omega_N^{jk}\\ &= \frac{1}{\sqrt{N}}\sum_{j\in\textrm{even}}x(j)\omega_N^{jk} + \frac{1}{\sqrt{N}}\sum_{j\in\textrm{odd}}x(j)\omega_N^{jk}\\ % &= \frac{1}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x(2j)\omega_N^{2jk} + \frac{1}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x(2j+1)\omega_N^{(2j+1)k}\\ % &= \frac{1}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x_E(j)\omega_N^{2jk} + \frac{1}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x_O(j)\omega_N^{2jk}\omega_N^{k} &[\because x_E,x_O]\\ % &= \frac{1}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x_E(j)\omega_{\frac N2}^{jk} + \frac{\omega_N^{k}}{\sqrt{N}}\sum_{j=0}^{\frac{N}{2}-1}x_O(j)\omega_{\frac N2}^{jk} &\left[\because \omega_N^2=\omega_\frac{N}{2}\right]\\ % &= \frac{1}{\sqrt{2\cdot\frac N2}}\sum_{j=0}^{\frac{N}{2}-1}x_E(j)\omega_{\frac N2}^{jk} + \frac{\omega_N^{k}}{\sqrt{2\cdot\frac N2}}\sum_{j=0}^{\frac{N}{2}-1}x_O(j)\omega_{\frac N2}^{jk}\\ % &= \frac{1}{\sqrt2}\frac{1}{\sqrt{\frac N2}}\sum_{j=0}^{\frac{N}{2}-1}x_E(j)\omega_{\frac N2}^{jk} + \frac{\omega_N^{k}}{\sqrt2}\frac{1}{\sqrt{\frac N2}}\sum_{j=0}^{\frac{N}{2}-1}x_O(j)\omega_{\frac N2}^{jk}\\ % &= \frac{1}{\sqrt2}X_E(k) + \frac{\omega_N^{k}}{\sqrt2}X_O(k)\\ % \end{align*} X(k)​=N​1​j=0∑N−1​x(j)ωNjk​=N​1​j∈even∑​x(j)ωNjk​+N​1​j∈odd∑​x(j)ωNjk​=N​1​j=0∑2N​−1​x(2j)ωN2jk​+N​1​j=0∑2N​−1​x(2j+1)ωN(2j+1)k​=N​1​j=0∑2N​−1​xE​(j)ωN2jk​+N​1​j=0∑2N​−1​xO​(j)ωN2jk​ωNk​=N​1​j=0∑2N​−1​xE​(j)ω2N​jk​+N​ωNk​​j=0∑2N​−1​xO​(j)ω2N​jk​=2⋅2N​​1​j=0∑2N​−1​xE​(j)ω2N​jk​+2⋅2N​​ωNk​​j=0∑2N​−1​xO​(j)ω2N​jk​=2​1​2N​​1​j=0∑2N​−1​xE​(j)ω2N​jk​+2​ωNk​​2N​​1​j=0∑2N​−1​xO​(j)ω2N​jk​=2​1​XE​(k)+2​ωNk​​XO​(k)​[∵xE​,xO​][∵ωN2​=ω2N​​]​

Thus NNN-point DFT can be expressed as a linear combination of two N2\frac N22N​-point DFT. There is actually one more optimization we can apply here. Recall that, xEx_ExE​ and xOx_OxO​ have periods of N2\frac N22N​. As such, XEX_EXE​ and XOX_OXO​ also have periods of N2\frac N22N​.

XE(k+N2)=XE(k)XO(k+N2)=XO(k) \begin{align*} X_E\left(k+\frac N2\right) &= X_E(k)\\ X_O\left(k+\frac N2\right) &= X_O(k) \end{align*} XE​(k+2N​)XO​(k+2N​)​=XE​(k)=XO​(k)​

Using these identities,
X(k+N2)=12XE(k+N2)+ωNk+N22XO(k+N2)=12XE(k)+ωNkωNN22XO(k)[∵period of N2]=12XE(k)−ωNk2XO(k)[∵ωNN2=−1] \begin{align*} X\left(k+\frac N2\right) &= \frac{1}{\sqrt2}X_E\left(k+\frac N2\right) + \frac{\omega_N^{k+\frac N2}}{\sqrt2}X_O\left(k+\frac N2\right)\\ &= \frac{1}{\sqrt2}X_E(k) + \frac{\omega_N^k\omega_N^\frac N2}{\sqrt2}X_O(k) &\left[\because\text{period of }\tfrac N2\right]\\ &= \frac{1}{\sqrt2}X_E(k) - \frac{\omega_N^k}{\sqrt2}X_O(k) &\left[\because\omega_N^{\frac N2}=-1\right]\\ \end{align*} X(k+2N​)​=2​1​XE​(k+2N​)+2​ωNk+2N​​​XO​(k+2N​)=2​1​XE​(k)+2​ωNk​ωN2N​​​XO​(k)=2​1​XE​(k)−2​ωNk​​XO​(k)​[∵period of 2N​][∵ωN2N​​=−1]​

Thus, to compute XXX we only have to iterate over kkk from 000 to (N2−1)\left(\frac N2-1\right)(2N​−1).

We may write a python program for the FFT algorithm as follows:

import numpy as np

def FFT(x):
  N = len(x)
  if N==1:
    return x
  w = np.exp(2j*np.pi/N)
  X_E = FFT(x[::2])
  X_O = FFT(x[1::2])
  X = [0]*N
  for k in range(N//2):
    X[k] = X_E[k]/np.sqrt(2) + X_O[k]*w**k/np.sqrt(2)
    X[k+N//2] = X_E[k]/np.sqrt(2) - X_O[k]*w**k/np.sqrt(2)
  return X

Quantum Fourier Transform

In QFT, we use an nnn-qubit system where N=2nN=2^nN=2n to denote the input vector ∣x⟩\ket{x}∣x⟩. The values are encoded in the amplitudes of the computational basis states. In other words, the jjj-th element of xxx was denoted as x(j)x(j)x(j) up until now; but in QFT it will be represented with ⟨j∣x⟩\braket{j|x}⟨j∣x⟩. We also, note that ∣X⟩=QFT∣x⟩\ket{X}=\textrm{QFT}\ket{x}∣X⟩=QFT∣x⟩ and,
∣x⟩=∣xE⟩+∣xO⟩∣xE⟩=∑j=0N2−1∣2j⟩ ⁣ ⁣⟨2j∣x⟩=∑j=0N2−1∣j⟩ ⁣ ⁣⟨j∣⊗∣0⟩ ⁣ ⁣⟨0∣x⟩∣xO⟩=∑j=0N2−1∣j⟩ ⁣ ⁣⟨j∣⊗∣1⟩ ⁣ ⁣⟨1∣x⟩ \begin{align*} \ket{x} &= \ket{x_E} + \ket{x_O}\\ \ket{x_E} &=\sum_{j=0}^{\frac N2-1}\ket{2j}\!\!\braket{2j|x} =\sum_{j=0}^{\frac N2-1}\ket{j}\!\!\bra{j}\otimes\ket0\!\!\braket{0|x}\\ \ket{x_O} &=\sum_{j=0}^{\frac N2-1}\ket{j}\!\!\bra{j}\otimes\ket1\!\!\braket{1|x} \end{align*} ∣x⟩∣xE​⟩∣xO​⟩​=∣xE​⟩+∣xO​⟩=j=0∑2N​−1​∣2j⟩⟨2j∣x⟩=j=0∑2N​−1​∣j⟩⟨j∣⊗∣0⟩⟨0∣x⟩=j=0∑2N​−1​∣j⟩⟨j∣⊗∣1⟩⟨1∣x⟩​
