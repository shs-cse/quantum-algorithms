Discrete Fourier Transform

Fast Fourier Transform

Directly computing DFT by multiplying the input vector with the DFT matrix has a time complexity of O(N2)O(N^2)O(N2). However, there is a more efficient algorithm commonly referred to as FFT. It is a divide and conquer algorithm and has a time complexity of O(Nlog⁡N)O(N\log N)O(NlogN). Let us discuss how FFT improves over DFT before we move onto QFT.

Here we are going to explain radix-2 decimation-in-frequency DIF-FFT algorithm1 in details. Simply because this one (rather than DIT-FFT) closely matches the traditionally used QFT circuit.

We may write a python program for the FFT algorithm as follows:

import numpy as np

def FFT(x):
  "Decimation in Frequency Fast Fourier Transform"
  N = len(x) # N = 2**n
  if N==1:
    return x
  w = np.exp(2j*np.pi/N)
  x_A = [(x[j] + x[j+N//2])/np.sqrt(2) for j in range(N//2)]
  x_B = [(x[j] - x[j+N//2])*w**j/np.sqrt(2) for j in range(N//2)]
  X = np.empty(N, dtype=complex)
  X[::2] = FFT(x_A)
  X[1::2] = FFT(x_B)
  return X

Quantum Fourier Transform

The Hadamard Part

In QFT, we use an nnn-qubit system where N=2nN=2^nN=2n to denote the input vector ∣x⟩\ket{x}∣x⟩. The values are encoded in the amplitudes of the computational basis states. In other words, up until now the jjj-th element of xxx was denoted as x(j)x(j)x(j). But in QFT it will be represented with ⟨j∣x⟩\braket{j|x}⟨j∣x⟩, and also, ∣X⟩=QFT∣x⟩\ket{X}=\textrm{QFT}\ket{x}∣X⟩=QFT∣x⟩. To build the quantum circuit for QFT, our strategy will be to just convert the butterfly diagram for the NNN-bit classical operations into nnn-qubit quantum gates.

Notice that the recursive butterfly unit has similarities to quantum gates. For example, we may consider the action of Hadamard gate on a single qubit. The Phase Factor Part

Now that we have figured out the Hadamard part already, only accounting for the phase factor part remains. This can be easily done with the help of phase gates. For example, in the N=23=8N=2^3=8N=23=8 case, as part of stage 2, we apply ωN21=ωN2\omega_\frac{N}{2}^1=\omega_N^2ω2N​1​=ωN2​ only on the states ∣011⟩≡∣3⟩\ket{011}\equiv\ket3∣011⟩≡∣3⟩ and ∣111⟩≡∣7⟩\ket{111}\equiv\ket7∣111⟩≡∣7⟩. Similarly, to introduce the phase factors at the end of stage 1, we can use only 2 different CP(θ)\textrm{CP}(\theta)CP(θ) gates. One of the gates will introduce ωN1\omega_N^1ωN1​ to both ∣101⟩≡∣5⟩\ket{101}\equiv\ket5∣101⟩≡∣5⟩ and ∣111⟩≡∣7⟩\ket{111}\equiv\ket7∣111⟩≡∣7⟩. We then again apply ωN2\omega_N^2ωN2​ to both ∣110⟩≡∣6⟩\ket{110}\equiv\ket6∣110⟩≡∣6⟩ and ∣111⟩≡∣7⟩\ket{111}\equiv\ket7∣111⟩≡∣7⟩. That way, ∣7⟩\ket7∣7⟩ will pickup ωN3\omega_N^3ωN3​ by conjunction.

To prove that this will work in the general case, we notice that in the recursive butterfly unit, we have ωNj\omega_N^jωNj​ applied on the basis vectors ∣j+N2⟩\Ket{j+\frac N2}∣∣​j+2N​⟩, where j∈ZN2j\in\Z_\frac{N}{2}j∈Z2N​​. That means, ωNj\omega_N^jωNj​ can be built from ωN1,ωN2,ωN4,ωN8\omega_N^1, \omega_N^2, \omega_N^4, \omega_N^8ωN1​,ωN2​,ωN4​,ωN8​ etc. since jjj can be expressed as a binary number an we apply the phase gate ωN2b\omega_N^{2^b}ωN2b​ if the bbb-th bit of jjj is 111. In the previous example we saw that ωN2\omega_N^2ωN2​ is applied to ∣6⟩\ket6∣6⟩ and ∣7⟩\ket7∣7⟩. This is because, if we equate ∣j+N2⟩\Ket{j+\frac N2}∣∣​j+2N​⟩ with either of them, we get j=2j=2j=2 and j=3j=3j=3. Both of them has 111 at bit index b=1b=1b=1; and so, we apply ωN2b=ωN21=ωN2\omega_N^{2^b}=\omega_N^{2^1}=\omega_N^{2}ωN2b​=ωN21​=ωN2​ to both of them.

Although we have justified that it is possible to breakdown ωNj\omega_N^jωNj​ into smaller phase gates; we have not proved that controlled phase gates are necessary. However, it is easy to see that we need controlled phase gates because we don’t want to apply phase gates on the basis vectors ∣j⟩\ket j∣j⟩ where j∈ZN2j\in\Z_\frac{N}{2}j∈Z2N​​. In order to do so, we need to set control on the MSB and then apply the phase gate on the target qubit. The distinction here between the control and target qubit is unnecessary since CP(θ)\textrm{CP}(\theta)CP(θ) gate is symmetric. For example, in the case of stage 1, we can take the nnn-th qubit as the control and take bbb-th qubit as the target qubit for a phase gate that applies the phase ωN2b\omega_N^{2^b}ωN2b​ if both qubits are in state ∣1⟩\ket1∣1⟩, otherwise does nothing.
